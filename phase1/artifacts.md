# Artifacts — one real-world text per day (README §0 artifact ritual)

Rules: attempt artifact N aloud at the start of day N (60 s), note the gap, do NOT
look anything up. Re-attempt as the day's last unit, THEN Pleco-check. Days 0–6 are
pinyin (the skill is tones/decoding); Day 7 on, hanzi — real Shenzhen/HIT-SZ text,
graded roughly to the vocab curve. Expect each artifact to be only PARTLY readable
on its day: the unreadable rest is next week's itch. Revisit any old artifact on a
deload day and feel the difference.
